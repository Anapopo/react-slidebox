## Install
+ use npm
```bash
npm install react-slidebox
```
+ use yarn 
```bash
yarn add react-slidebox
```

## Use
```javascript
// For ES6
import ReactSlideBox from 'react-slidebox'

//Then use <ReactSlideBox {...props} /> like Other React Component

```
## Build
`npm run build` or `yarn build`
